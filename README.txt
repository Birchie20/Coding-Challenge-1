To run the application please follow the below steps

1) Ensure that all the data required is stored in the data folder within the solution (this currently houses the test data)
2) Create a new virtual environment and install the libraries found requirements/requirements.text
3) Activate this environment
4) Run web_form.py from your terminal
5) Terminal will provide you with the address to go to in your browser to view the app locally (i.e. http://127.0.0.1:5000/)
6) Use the instructions provided to produce a report
